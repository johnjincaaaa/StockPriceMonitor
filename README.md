
# StockPriceMonitor 股票价格实时监控系统

## 项目简介

StockPriceMonitor是一个股票价格实时监控工具，能够实时获取股票价格、涨跌幅、主力净流入等数据，并支持设置价格提醒功能，帮助用户及时掌握股票动态。

## 功能特点

- 实时监控股票价格、涨跌幅数据
- 跟踪主力资金净流入情况
- 支持添加/移除监控股票
- 自定义价格提醒设置
- 记录监控日志和提醒历史
- 直观的图形化用户界面

## 核心功能模块

1. **数据获取模块**：通过网络接口获取股票实时数据
   - 股票基本信息（名称、最新价、涨跌幅等）
   - 主力资金净流入数据

2. **监控模块**：后台线程

3. **提醒模块**：基于用户设置的阈值触发提醒
4. **用户界面**：直观的操作界面，展示股票列表和提醒日志

## 数据来源

系统通过东方财富网相关接口获取股票数据，包括：
- 股票实时价格和涨跌幅
- 主力资金流向数据

## 文件结构说明

- `main.py`：程序主入口，包含UI界面和核心逻辑
- `get_data.py`：股票基础数据获取模块
- `fund_flow.py`：主力资金流向数据获取模块
- `stocks.json`：保存监控的股票列表及数据
- `alert_settings.json`：保存用户设置的提醒条件
- `stock_monitor.log`：系统运行日志文件

## 使用方法

1. 启动程序后，默认加载预设的股票列表
2. 点击"添加股票"按钮可添加新的监控股票（需输入正确的股票代码，如sh600036）
3. 选中股票后，点击"移除股票"可删除监控项
4. 点击"设置提醒"可针对选中股票设置价格或资金流向提醒条件
5. 点击"开始监控"启动实时数据更新
6. 提醒信息会显示在下方的提醒日志区域

## 注意事项

- 程序需要网络连接以获取实时数据
- 股票代码需遵循"sh"（沪市）或"sz"（深市）前缀格式
- 频繁的数据请求可能受到接口限制，程序已内置重试机制
